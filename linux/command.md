# basic command

> 리눅스 기본 명령어



### 디렉토리

##### cd (change directory)

- 폴더를 이동하는 명령어
- `cd  <directory name>`



##### pwd (print working directory)

- 현재 디렉토리 위치 확인



##### mkdir (make directory)

- 폴더를 생성하는 명령어
- `mkdir <directory name>`



### 파일

##### ls (list)

- 파일 목록 확인
- `ls -a` 옵션으로 숨겨진 파일, 폴더까지 출력할 수 있음



##### touch

- 파일을 생성
- `touch <file name>`
